# 🤖 AI Interview Assistant

An AI-powered web application that helps users practice technical interviews using their resume.
Upload your resume, get AI-generated questions, answer them, and receive feedback & score.

---

## 🚀 Features

* 📄 Resume Upload (PDF)
* 🤖 AI-generated interview questions
* 🎤 Voice input support
* 💬 Chat-based interview UI
* 🧠 AI evaluation & scoring
* 📊 Feedback with improvements & tips
* 💾 Auto-save interview progress
* 🌙 Dark / Light mode

---

## 🛠 Tech Stack

### Frontend

* React (Vite)
* Tailwind CSS
* Axios

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)
* Multer (file upload)

### AI

* OpenRouter API

## ⚙️ Environment Variables

Create a `.env` file in backend:

```
MONGO_URI=your_mongodb_connection
OPENROUTER_API_KEY=your_api_key
JWT_SECRET=your_secret
PORT=880
```

For frontend:

```
VITE_API_URL=https://your-backend-url.onrender.com/api
```

## 📦 Installation (Local Setup)

### 1️⃣ Clone repo

```
git clone https://github.com/your-username/AI-Interview-Assistant.git
cd AI-Interview-Assistant
```

---

### 2️⃣ Backend setup

```
cd backend
npm install
npm run dev
```

---

### 3️⃣ Frontend setup

```
cd frontend
npm install
npm run dev
```

---

## 🚀 Deployment

### Backend (Render)

* Add Environment Variables in Render dashboard
* Start Command: `node server.js`

### Frontend (Render)

* Add `VITE_API_URL`
* Build Command: `npm run build`
* Publish Directory: `dist`
